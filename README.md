# MyAdmin_2 - Cool Admin Management System

A premium Django-based management system for educational institutions featuring a modern "Cool Admin" design and full CRUD functionality.

## 🚀 Features

- **Modern UI**: Clean, white sidebar with dark theme elements, inspired by professional admin dashboards.
- **Comprehensive Management**:
  - **Faculty**: Add and manage university faculties.
  - **Kafedra**: Manage specific departments/chairs.
  - **Subject**: Track and manage subjects taught.
  - **Teacher**: Full teacher management with subject associations.
  - **Group**: Organize students into groups linked to faculties.
  - **Student**: Comprehensive student database management.
- **Full CRUD**: Create, Read, Update, and Delete capabilities for all modules.
- **Responsive Design**: Fully functional on desktop and mobile devices.

## 🛠 Tech Stack

- **Backend**: Django 6.x
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Icons**: FontAwesome 5
- **Database**: SQLite (Development)

## 💻 Installation & Setup

1. **Clone the repository**:

   ```bash
   git clone https://github.com/DasturchiMadaminjon/my_admin_site_2.git
   cd my_admin_site_2
   ```

2. **Apply Migrations**:

   ```bash
   python manage.py migrate
   ```

3. **Run the Server**:

   ```bash
   python manage.py runserver 8001
   ```

4. **Access the Application**:
   Open your browser and navigate to `http://127.0.0.1:8001/`
